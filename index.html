US Flight Delay Dashboard

This project features a comprehensive visualization of flight delays in the United States. Using Tableau, I created an interactive dashboard that allows users to analyze delay patterns based on various factors such as airline, airport, season, and time of day. Key features include heat maps to show areas with the highest delays, time trend graphs, and performance comparisons between airlines. This dashboard helps stakeholders in the aviation industry to identify areas of improvement and optimize flight schedules.

Supermarket Sales Dashboard

Using Power BI, I developed a dashboard that depicts an in-depth analysis of supermarket sales. The dashboard includes key visualizations such as monthly sales trends, product performance by category, profitability analysis, and customer segmentation. I also implemented a drill-down feature that allows users to explore data from the macro level to individual transaction details. The dashboard helps supermarket management make strategic decisions regarding inventory, promotions, and customer service improvements.

GUI Application for Predicting Stock Prices of LQ45 Index
